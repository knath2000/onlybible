## Responsive Layout Patterns

### Desktop Layout (â‰¥1024px)
- **Topbar**: 3-zone horizontal layout
- **Content**: Max-width centered, generous margins
- **Verse Cards**: Full-width with spacious padding
- **Navigation**: All controls visible

### Tablet Layout (768px - 1023px)
- **Topbar**: Condensed zones, smaller gaps
- **Content**: Reduced max-width
- **Verse Cards**: Maintained padding
- **Navigation**: Pill selectors still accessible

### Mobile Layout (<768px)
- **Topbar**: Stacked zones, simplified
- **Content**: Full-width with minimal margins
- **Verse Cards**: Reduced padding
- **Navigation**: Touch-friendly sizing

## Performance Considerations

### Layout Stability
- **Avoid layout shifts**: Reserve space for dynamic content
- **Stable anchors**: Consistent positioning for alignment overlays
- **Progressive loading**: Infinite scroll without layout jumps

### Animation Performance
- **GPU acceleration**: Transform and opacity properties
- **Avoid layout triggers**: No width/height animations
- **Reduced motion**: Respect user preferences
- **Frame rate**: 60fps target for smooth interactions
