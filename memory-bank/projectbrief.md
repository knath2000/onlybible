# Project Brief: Bible Reading App with Translation

## Project Overview
A Next.js web application for reading the Bible in Spanish (Reina-Valera version) with translation capabilities to English for both verses and individual words.

## Core Objectives
1. Provide easy access to Spanish Bible content
2. Enable quick translation of verses from Spanish to English
3. Support word-level translation for language learning
4. Create a beautiful, modern glassmorphic UI
5. Ensure responsive design for all devices

## Target Audience
- Spanish-speaking Christians
- Bible students learning Spanish
- Language learners using Bible content
- Pastors and teachers preparing materials

## Key Features
- Spanish Bible reading (Reina-Valera version)
- Verse translation (Spanish to English)
- Word-level translation with tooltips
- Chapter and verse navigation
- Glassmorphic UI design
- Responsive layout
- Caching for performance
- Error handling and loading states

## Success Metrics
- User can navigate to any Bible verse
- Translation works for both verses and words
- UI is visually appealing and functional
- App performs well with caching
- Responsive design works on mobile and desktop

## Technical Stack
- Next.js 16 (App Router)
- React with TypeScript
- Tailwind CSS
- Bible API (bible-api.com)
- Mock translation service (replaceable)
- Glassmorphic design system
